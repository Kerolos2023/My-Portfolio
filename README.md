# Portfolio Website
---

## Project Overview
This is a **personal Portfolio Website** built with **React**, **Bootstrap**, **HTML**, **CSS**, and **JavaScript**.  

The portfolio showcases my **projects, skills, and experience**. It is fully responsive, interactive, and visually appealing.  

---

## Features
- Responsive design for desktop, tablet, and mobile
- Sections for About Me, Skills, Projects, and Contact
- Interactive UI components using **MUI** and **Bootstrap**
- Smooth scrolling and modern animations
- Contact form (can be connected to backend later)
- Clean, professional, and modern design

---

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript, React  
- **UI Libraries:** Bootstrap 5, Material UI (MUI)  
- **Version Control:** Git & GitHub
